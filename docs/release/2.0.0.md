## v2.0.0

Major release - Adds the remaining portfolio sections (Projects, Contact, Footer), a full backend API layer with
serverless functions, and a comprehensive icon system.

---

### New Pages & Sections

- **Projects** — New section showcasing project categories with an interactive `SpotlightText` component that follows
  the user's mouse cursor.
- **Contact** — Full contact page with floating label fields, client-side validation, animated submit button with
  success/error status icons, and backend API integration via Netlify Functions.
- **Footer** — Site footer with social links (GitHub, LinkedIn, X, Instagram, Spotify, Email) and hover color
  transitions.

### New Components

- **`Tape`** — Infinite marquee tape component with left/right scroll directions, scaled up for large desktop viewports.
- **`SpotlightText`** — Interactive text component with a mouse-following radial spotlight effect.
- **`FloatingField`** — Floating label input/textarea fields with validation error display and reserved error
  placeholder space.
- **`SubmitButton`** — Animated submit button with draw-check (success) and shake (error) status icon animations.
- **`ResumeDropdown`** — Dropdown button linking to resume PDFs in English and Spanish, with click-outside and
  Escape-to-close behavior.
- **`ScrollHint`** — Animated scroll indicator with idle timing logic, breakpoint guard, and hero section awareness.
- **`SvgIcon`** — Refactored icon system with `currentColor` support, centralizing all icons through a single component
  with a declarative `icon` prop.

### New Hooks

- **`useContactForm`** — Form state management for the contact page with field validation, server error mapping,
  auto-reset status timer, and analytics event tracking.
- **`useMousePosition`** — Mouse position tracking hook for the spotlight text effect.
- **`useScrollHint`** — Controls scroll hint visibility with idle detection timing.

### Backend — Netlify Serverless Functions

- **`/api/contact`** — POST endpoint for sending contact form emails via Resend.
- **`/api/subscribers`** — POST endpoint for newsletter email subscriptions via Resend Audiences.
- **Email sender service** — Resend-based email client with daily quota detection (429) and user-friendly
  time-until-reset messages.
- **Error handler** — `ApiError` class and `withErrorHandler` middleware for consistent JSON error responses.
- **Validation schemas** — Valibot schemas for contact and subscriber request validation with custom error messages.
- **Middleware composer** — Utility to chain Netlify Function middlewares.
- **Integration & unit tests** — Full test coverage for all functions, middlewares, schemas, utils, and the email
  sender.

### Icons

- Added 11 new SVG icons: `ArrowDown`, `ChevronDown`, `Email`, `Error`, `Github`, `Instagram`, `Linkedin`,
  `OpenNewWindow`, `Spotify`, `Success`, `X`.
- Added `build:icons` script to auto-generate React components from SVG files via SVGR.

### Infrastructure & Config

- **Dependencies** — Added `resend`, `valibot`, and `@netlify/functions`.
- **Environment variables** — Replaced `VITE_` prefix with explicit `define` injection in Vite config. Added social link
  env vars and `.env.example`.
- **Security** — Added `.npmrc` with npm supply chain protections.
- **SonarQube** — Extended coverage to include the `netlify/` folder.
- **Netlify** — Configured Functions directory and updated `tsconfig` path aliases for the serverless layer.

### Improvements & Refactors

- Migrated all inline `CSSProperties` to `ResponsiveStyles` across existing components for consistent responsive
  behavior.
- Restructured content constants into a centralized `content.tsx` with typed exports (`HEADER`, `HERO`, `ABOUT`,
  `PROJECTS`, `CONTACT`, `FOOTER`).
- Refactored analytics `track()` to accept a structured properties object.
- Added resume PDF files (English and Spanish) to `public/files/`.
- Fixed About section GSAP mock in tests and expanded overall test coverage with new tests for `Footer`, `ScrollHint`,
  `useScrollHint`, `ResumeDropdown`, `Projects`, `SpotlightText`, `useMousePosition`, `HeroScene`, `Contact`,
  `FloatingField`, `SubmitButton`, and `useContactForm`.
