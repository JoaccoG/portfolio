## v1.0.0

Major release - Initial release of [joaquingodoy.com](https://joaquingodoy.com), my professional portfolio, built with
TypeScript, Vite, React, and deployed on Netlify.

---

### Pages & Sections

- **Hero** — Animated landing section with a 3D interactive scene (React Three Fiber), a status badge indicating
  availability, and rotating personality traits.
- **About** — Multi-chapter storytelling section with scroll-triggered GSAP animations, and a "Full Resume" dropdown
  linking to PDF resumes in English and Spanish.
- **Header** — Fixed navigation bar with scroll-shrink animation, responsive nav links, and a "Blog" link with an SVG
  underline draw animation.

### Visual Overlay System

- **Vignette layer** — Fullscreen vignette effect via custom GLSL fragment shader.
- **Grain layer** — Film grain post-processing effect with an optimized `GrainRenderer` class and LCG-based
  pseudo-random generator.
- **Grid layer** — Animated orb particle system with debounced respawn on resize and viewport-based wrapping, rendered
  via instanced meshes and custom shaders.
- **FullscreenQuad** — Shared geometry component for fullscreen shader passes.

### Core Hooks

- **`useBreakpoint`** — Responsive breakpoint detection with a `resolve()` function overload for declarative
  per-breakpoint style values.
- **`useSmoothScroll`** — Bridges GSAP ScrollTrigger with Lenis for smooth, inertia-based scrolling.
- **`useOrbSimulation`** — Physics simulation hook driving the grid layer orb system.

### Components & Shared UI

- **`Section`** — Layout wrapper for page sections with consistent spacing.
- **`Title`** — Animated section title component.
- **`SvgIcon`** — Icon system powered by SVGR, generating React components with ref support from SVG files.

### Content & Styling

- Global CSS variables, resets, and base typography (`Bebas Neue`, `Fira Code`, `Inter`).
- Keyframe animations for the status badge pulse.
- Centralized content constants (`HEADER`, `HERO`, `ABOUT`).
- SEO setup with `robots.txt` and `sitemap.xml`.

### Analytics

- **Umami** integration with async script loading, event queue for pre-load tracking, and a `track()` utility function.

### Infrastructure & Tooling

- **Build** — Vite 7 with SWC + ESBuild, TypeScript path aliases, pnpm workspaces.
- **Testing** — Vitest with coverage (v8), Testing Library (React + user-event), React Three Test Renderer, custom test
  helpers for env/media/SVG mocking.
- **Linting** — ESLint 9 flat config + Prettier + typescript-eslint + Stylistic plugin.
- **Git hooks** — Husky with `commit-msg` (conventional commit validation), `pre-commit` (lint-staged), and `pre-push`
  (type-check + tests).
- **CI/CD** — GitHub Actions workflows for audit, build, and test. SonarQube integration for code quality.
- **Deployment** — Netlify with SPA redirect, configured Node.js and pnpm versions.
- **Repo quality** — `.editorconfig`, `.gitattributes`, PR template, Copilot instructions, MIT license, FUNDING.yml.
